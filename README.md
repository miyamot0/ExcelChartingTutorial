# ExcelChartingTutorial
This repository holds all of the example spreadsheets used in the "Intro to Charting in the Behavioral Sciences" youtube tutorial.

Charts modeled include:
  - Simple line
  - Complex line
  - Simple bar
  - Hybrid plot
  - Multi-plot (multiple baseline)
